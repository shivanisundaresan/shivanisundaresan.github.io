My favorite coding projects from school:

Insta485 (EECS 485, November 2023)

Created a fully functional clone of Instagram with client-side and sever-side dynamic pages.

LC2K simulator (EECS 370, April 2023)

Created a pipeline simulator for the Little Computer 2000 assembly language.

Computer Security Forensics project (EECS 388, March 2023)

Performed cryptography, website, network and buffer overflow (assembly) attacks to perform a forensic analysis on an employee's hard drive.

AmongUs simulator (EECS 281, December 2022)

Created a simulation of the popular AmongUs game using different solutions to the travelling salesman problem (FastTSP, OptTSP, MST, etc).

Man vs Zombie simulator (EECS 281, October 2022)

Used different types of priority queues (pairing, binary, unordered, sorted) to determine the order in which to eliminate the opponent zombies of varying health levels, distance, and species.
